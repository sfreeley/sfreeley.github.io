Front-end Capstone Project

Inspired by my experience as a pharmacist, this single page application sought to help alleviate some of the confusion that comes with taking multiple medications. Users can:

- Document the drug name, dosage, dosage form, patient-specific directions, indication, and notes all in one place.
- Input prescription numbers and refills (if applicable) as well as last fill date and days supply, which will then automatically calculate their next refill date.
- Track next refill dates for each medication with a real-time countdown (by days), which will also be displayed to prevent users from running out of their medication.
- Utilize complete CRUD functionality and perform each of these functions on each medication entry.
- Search specific keywords pertaining to their medications (ie name, dosage, dosage form, indication, etc)
- Select pertinent health resource topics through the use of health.gov external API
